# AI Coding kit

这是我在实际开发中使用的 **skills 与 MCP 配置** 的集合，用于沉淀和复用 AI 辅助编程能力。

## AI Coding 规范

## Skills

### Install Skills

- 使用 skills 安装 `pnpm dlx skills add Pony-Unicorn/ai-coding-kit`
- gemini 安装 `gemini skills install .agents/skills --scope workspace`

### Use Skills

- [ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill?tab=readme-ov-file#using-cli-recommended)
  - 专业UI/UX设计，需要单独命令行安装
- `git-commit` 提交 git 助手
  - remove Claude co-author 在 settings.json 中设置 `{"attribution": {"commits": false,"pullRequests": false}}`
- `code-review` TypeScript、Python、Swift、Kotlin、Go的全面代码审查技能。
- `security-review` 安全检查在添加身份验证、处理用户输入、处理机密信息、创建API端点或实现支付/敏感功能时使用此技能
- `typescript-patterns` typescript 最佳实践
- `go-patterns` go 最佳实践
- `database-design` 数据库设计
